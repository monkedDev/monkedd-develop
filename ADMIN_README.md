# 🎉 Веб-панель администратора (без сервера!)

Полностью клиентская админ-панель — работает **без Vercel, без сервера, без базы данных**!

---

## 🚀 Как это работает

- ✅ Заявки хранятся в **localStorage** браузера
- ✅ Админ-панель работает прямо на **GitHub Pages**
- ✅ Не нужен сервер или API
- ✅ Полностью бесплатно

## 🔒 Безопасность

⚠️ **Важно:** Это клиентское решение, поэтому:

- ❌ Пароль хранится в localStorage (не для продакшена)
- ❌ Данные доступны только в твоём браузере
- ❌ При очистке кэша данные пропадут

**Для продакшена** используй серверное решение (Vercel + база данных).

---

## 📝 Как пользоваться

### 1. Клиент заполняет форму

На главной странице клиент заполняет форму заказа.

### 2. Заявка сохраняется

Данные сохраняются в localStorage того браузера, где была заполнена форма.

### 3. Ты заходишь в админку

Открываешь `/admin/` и видишь все заявки.

### 4. Управляешь заявками

- Меняешь статусы
- Просматриваешь детали
- Экспортируешь в CSV

## ⚙️ Настройки

Открой `js/admin.js` и найди:

```javascript
const ADMIN_PASSWORD = 'monkeddev2025';
```

Поменяй на свой пароль.

### Очистить все заявки

В консоли браузера выполни:

```javascript
localStorage.removeItem('monkeddev_orders');
location.reload();
```

## ⚠️ Ограничения

1. **Данные в браузере** — если клиент заполнит форму на одном устройстве, ты увидишь заявку только на этом же устройстве.

2. **Нет синхронизации** — localStorage не синхронизируется между устройствами.

3. **При очистке кэша** — данные пропадут.

**Решение:** Для реальной работы используй серверное решение с базой данных.
