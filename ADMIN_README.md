# 📧 Админ-панель monkedDev + Web3Forms

Заявки отправляются через **Web3Forms** на почту **monkeddev@inbox.ru**

Админ-панель показывает демонстрационные заявки для примера управления.

---

## 📋 Как работает

1. **Клиент заполняет форму** на сайте
2. **Заявка приходит тебе на почту** monkeddev@inbox.ru
3. **Ты отвечаешь клиенту** напрямую через email/Telegram/телефон

---

## 🚀 Настройка Web3Forms

### 1. Получи access key

1. Зайди на [web3forms.com](https://web3forms.com)
2. Введи свой email (monkeddev@inbox.ru)
3. Получи access key на почту

### 2. Вставь key в форму

В `index.html` найди форму и вставь свой key:

```html
<input type="hidden" name="access_key" value="ТВОЙ_КЛЮЧ">
```

### 3. Готово!

Теперь заявки будут приходить на почту.

---

## 📊 Админ-панель

### Вход

Открой: `https://monkeddev.github.io/monkedd-develop/admin/`

**Пароль:** `monkeddev2025`

### Функции

- 📊 **Дашборд** — статистика и последние заявки
- 📋 **Заявки** — просмотр, фильтры, поиск
- ✏️ **Статусы** — Новая, В работе, Завершена, Отменена
- 📥 **Экспорт** — выгрузка в CSV
- 📈 **Аналитика** — графики по дням и услугам

⚠️ **Важно:** Админ-панель показывает демонстрационные данные. Реальные заявки приходят только на почту!

---

## 📬 Как обрабатывать заявки

1. **Пришло письмо** на monkeddev@inbox.ru
2. **Открыл письмо** — там все данные клиента:
   - Имя
   - Контакт (email/Telegram/телефон)
   - Тип проекта
   - Бюджет
   - Сообщение
3. **Связался с клиентом** через указанный контакт
4. **Обсудил детали** и начал работу

---

## 🔒 Безопасность

- ✅ Web3Forms обрабатывает данные безопасно
- ✅ Пароль от админки хранится в localStorage
- ✅ Можно сменить пароль в `js/admin.js`

---

## ⚙️ Настройки

### Сменить пароль админки

В `js/admin.js` найди:

```javascript
const ADMIN_PASSWORD = 'monkeddev2025';
```

Поменяй на свой.

### Очистить демо-заявки

В консоли на вкладке админки:

```javascript
localStorage.removeItem('monkeddev_orders_demo');
location.reload();
```
