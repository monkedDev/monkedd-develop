---

## 🚀 Установка

### Шаг 1: Задеплой на Vercel

```bash
# Установи Vercel CLI (если нет)
npm i -g vercel

# Залогинься
vercel login

# Задеплой
vercel --prod
```

Или через сайт:
1. Зайди на [vercel.com](https://vercel.com)
2. **New Project** → выбери `monkedd-develop`
3. **Deploy**

### Шаг 2: Настрой переменные окружения

В настройках проекта на Vercel добавь:

- `ADMIN_PASSWORD` = твой пароль для входа в админку (по умолчанию: `monkeddev2025`)

### Шаг 3: Готово!

Теперь у тебя есть:
- ✅ Форма на сайте принимает заявки
- ✅ Заявки хранятся в API
- ✅ Админ-панель для управления

## 📊 Как пользоваться

### 1. Вход в админку

Открой: `https://твой-сайт.vercel.app/admin/`

Пароль по умолчанию: **monkeddev2025**

### 2. Дашборд

- 📊 Статистика по заявкам
- 📋 Последние 5 заявок
- ⚡ Быстрый доступ

### 3. Управление заявками

- 👁️ Просмотр деталей
- ✏️ Изменение статуса
- 🗑️ Удаление
- 🔍 Поиск и фильтры
- 📥 Экспорт в CSV

### 4. Статусы заявок

- 🆕 **Новая** — только что полученная
- ⏳ **В работе** — обрабатываешь
- ✅ **Завершена** — проект сделан
- ❌ **Отменена** — клиент передумал

---

## 🔒 Безопасность

- ✅ Пароль хранится в localStorage
- ✅ API защищено CORS
- ✅ Авторизация через Bearer токен
- ✅ Honeypot от спама

**Важно:** Поменяй пароль в `api/orders.js` или через переменную `ADMIN_PASSWORD`!

---

## 📝 Как работает

1. Клиент заполняет форму на сайте
2. Данные отправляются на `/api/orders`
3. Заявка сохраняется в API
4. Ты видишь заявку в админке
5. Меняешь статус → связываешься с клиентом

## 🎨 Кастомизация

### Поменять пароль

В `api/orders.js` найди:
```javascript
const adminPassword = process.env.ADMIN_PASSWORD || 'monkeddev2025';
```

Или задай переменную `ADMIN_PASSWORD` в Vercel.

### Изменить статусы

В `js/admin.js` найди функцию `renderStatus` и добавь свои.

### Добавить email уведомления

В `api/orders.js` после сохранения заявки добавь отправку email через SendGrid или Resend.
