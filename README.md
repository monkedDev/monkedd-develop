# monkedDev — Сайт-портфолио

Сайт-визитка для приёма заказов на разработку.

## 📁 Структура проекта

```
monkeddev-site/
├── index.html          # Главная страница
├── css/
│   └── style.css       # Все стили
├── js/
│   └── main.js         # JavaScript (форма, анимации, меню)
└── assets/             # Папка для изображений (опционально)
```

## 🚀 Как запустить

Просто открой `index.html` в браузере (дважды кликни по файлу).

## ⚙️ Настройка отправки формы (Web3Forms)

Форма настроена на **Web3Forms** — **полностью бесплатно и безлимитно**.

### Быстрая настройка (2 минуты):

1. **Зайди на [web3forms.com](https://web3forms.com)**
2. **Введи свой email** (`monkeddev@inbox.ru`) в поле на главной
3. **Нажми "Get API Key"**
4. **Проверь почту** — придёт письмо с твоим access_key
5. **Вставь ключ в index.html:**
   - Открой `index.html`
   - Найди строку: `<input type="hidden" name="access_key" value="...">`
   - Замени значение на свой ключ из письма
6. **Готово!**

### Первый тест:

1. Открой сайт в браузере
2. Заполни форму тестовыми данными
3. Нажми «Отправить»
4. Проверь почту — должно прийти письмо с заявкой

### Преимущества Web3Forms:

- ✅ **Полностью бесплатно** — никаких лимитов
- ✅ **Безлимитно** — сколько угодно писем
- ✅ **Без регистрации** — только email для получения ключа
- ✅ **Надёжно** — письма не попадают в спам
- ✅ **Без сервера** — работает на статических сайтах
